### II. DESCRIPCIÓN DE LA INVENCIÓN

El suscripto, **Sebastián Maximiliano Monteleón**, DNI N° 32.725.461, declara bajo juramento ser el **autor, diseñador e inventor original** del sistema denominado **uni.id**, cuya concepción y desarrollo conceptual se describe a continuación:

#### A. Nombre del sistema
**uni.id** — Sistema de Identidad Digital Distribuida

#### B. Concepto central
Un sistema donde **cada persona es un nodo de identidad digital** dentro de una red distribuida global. El sistema permite que cualquier persona almacene, gestione y comparta su identidad digital de forma segura, descentralizada y criptográficamente verificable.

#### C. Componentes del sistema (concebidos por el autor)

1. **Wallet de Identidad (uni.id)**
   - Cada usuario posee una billetera de identidad digital única
   - Almacena documentos personales: DNI, estudios, historial médico, licencia de conducir, escrituras de propiedades, registro de mascotas y otros
   - Identidad criptográfica propia e intransferible

2. **Red de Identidad Distribuida**
   - Cada persona es un nodo dentro de la red
   - Los nodos se conectan formando una red de confianza descentralizada
   - La red no tiene un punto central de control — es un ecosistema global

3. **Capa de Seguridad Adaptativa — Sistema Inmunológico Digital**
   - La red aprende y se protege sola
   - Se comporta como un sistema vivo detectando anomalías
   - Cuando se detecta un ataque: la red aprende el patrón, protege otros nodos automáticamente, y la seguridad mejora continuamente
   - El sistema completo se refuerza a sí mismo sin intervención humana

4. **Infraestructura Global**
   - Capa de infraestructura distribuida sobre la red
   - Cifrado de extremo a extremo
   - Ecosistema de servicios para terceros (bancos, escuelas, hospitales, aeropuertos, gobierno, inmobiliarias)

5. **Ecosistema de Servicios**
   - El sistema se convierte en una **capa de identidad digital sobre internet**
   - Instituciones que pueden integrarse: bancos, escuelas, hospitales, aeropuertos, organismos de gobierno, inmobiliarias, entre otros
   - Resultado: una identidad digital universal, verificable y segura para cada persona en el mundo

#### D. Arquitectura conceptual (en orden de capas)

```
USUARIO
  └── Wallet uni.id / Identidad Humanidad
        └── Red de Identidad (nodos distribuidos)
              └── Capa de Seguridad Adaptativa
                    └── Infraestructura Global
                          └── Ecosistema de Servicios
```
